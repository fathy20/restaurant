# خطة التنفيذ: تحسينات موقع مطعم الضيافة

## نظرة عامة

تحويل موقع مطعم الضيافة إلى موقع عربي بالكامل مع تحسينات SEO وأداء شاملة. المشروع يستخدم React + TypeScript + Vite ويتضمن تحديث ستة مكونات رئيسية، إضافة دعم RTL، تحسين الصور، وإزالة وظائف تبديل اللغة.

## ملاحظات

- المهام المميزة بـ `*` اختيارية ويمكن تخطيها للحصول على MVP أسرع
- كل مهمة تشير إلى المتطلبات المحددة للتتبع
- Checkpoints تضمن التحقق التدريجي
- Property tests تتحقق من الخصائص العامة للصحة
- Unit tests تتحقق من أمثلة محددة وحالات حدية
- جميع الاختبارات تستخدم Vitest + @testing-library/react + fast-check
- Property tests يجب أن تعمل على الأقل 100 iteration

## متطلبات التشغيل

```bash
# تشغيل جميع الاختبارات
npm run test

# تشغيل الاختبارات في وضع المراقبة
npm run test:watch

# تشغيل الاختبارات مع تقرير التغطية
npm run test:coverage

# تشغيل Property tests فقط
npm run test -- --grep "Property"

# تشغيل اختبارات Accessibility فقط
npm run test -- --grep "Accessibility"
```
